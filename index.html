First Parameter

/date will display only the date.

/time will display only the date.

/datetime will display both date and time.

/custom will allow your to specify your own format

Second Parameter

nothing or /html will return request as HTML

/json will return request as JSON

/xml will return request as XML

/plain will return request as plain text

Custom Format Codes

Type a custom format after /custom to receive custom format.

Examples

/date/json will return 'Mon 05-16-2016' in JSON format

/datetime will return '13:00:00' (1pm military) in HTML

/time/plain will return "13:00:00' as plain text

/custom/MM-YY - will return '05-16' for May 2016 in HTML

/custom/MM YY/xml - will return '05 16' for May 2016 in XML
