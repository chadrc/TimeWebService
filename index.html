First Parameter

/date will display only the date.

/time will display only the date.

/datetime will display both date and time.

Second Parameter

nothing or /html will return request as HTML

/json will return request as JSON

/xml will return request as XML

/plain will return request as plain text

Examples

/date/json will return only the date in JSON format

/datetime will return the data and time in HTML

/time/plain will return the time as plain text
